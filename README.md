COME BUY 🚀

A futuristic, AI-powered eCommerce hardware store built with HTML, CSS, and vanilla JS. Features dark UI, neon colors, glassmorphism, 3D animated cards, search, checkout flows, and a simulated backend runtime!

## 🌐 Live Website

Click the link below to open the live store directly in your web browser:
👉 **[Open Anti-Gravity Store]*https://mugilan38.github.io/come-by/* 👈

*(Note: The link above assumes GitHub Pages is enabled on the `main` branch).*

## ✨ Features

- **Futuristic Design**: Dark styling with neon gradient text and glow effects.
- **AI Recommendations**: Products are dynamically sorted on the homepage natively.
- **Product Catalog**: Fully featured grid layout with side-bar filtering and a functional search bar.
- **Functioning Shopping Cart**: Utilizing LocalStorage to preserve your cart data.
- **Checkout Flow**: Complete interactive layout handling payment routing and clearing carts smoothly.
- **Admin Dashboard**: Contains a mock analytics command center utilizing Chart.js.

## 🛠️ Built With

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Charting**: Chart.js (for the Admin UI)
- **Data Layer**: Mock JSON database handling simulated API fetches.

## 💻 Local Testing (Without Node)

Because this app utilizes a client-side architecture to simulate an API:
1. Clone the repository `git clone https://github.com/Mugilan38/come-by.git`
2. Open `public/index.html` directly in your browser. No server environment is required!
